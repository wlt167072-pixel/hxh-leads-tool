🌍 第一步：全球搜索（怎么找客户）
1️⃣ 切换到「全球搜索」Tab
2️⃣ 在国家网格里点击目标国家（如 Nigeria、Ghana）
3️⃣ 点「🔍 搜索选定国家的客户」— 生成 Google/LinkedIn 搜索链接
4️⃣ 在页面底部的 SerpAPI 全网搜索模块里输入关键词自动搜索
5️⃣ 点搜索结果的「⬇️ 导入为客户」按钮
💡 SerpAPI 设置：去 serpapi.com 免费注册，获取 API Key（100次/月免费），粘贴到 SerpAPI 模块保存即可。
📞 第二步：找联系方式（3种方法）
方法① SerpAPI 自动识别邮箱（最简单）
搜索时勾选「识别邮箱」，在结果里直接点蓝色邮箱地址复制。
方法② Hunter.io API 批量查询（最全）
在「联系方式」Tab 填入 Hunter.io Key → 点「一键搜索全部客户邮箱」
方法③ Apollo + Chrome 插件（最准）
Apollo.io 上找到联系人 → HXH 插件自动保存邮箱+电话 → 一键填入
✉️ 第三步：发开发信（AI 生成）
1️⃣ 在「开发信」Tab，从下拉选择一个客户
2️⃣ 选择语言（英文/中文）和邮件类型
3️⃣ 点「✨ 生成开发信」→ AI 自动写好邮件
4️⃣ 复制内容，在 Gmail 里粘贴发送
5️⃣ 回到「全部客户」→ 点客户卡片 → 在「跟进」区域记录发送状态
📱 也可以发 WhatsApp：在「联系方式」Tab 点「📱 批量 WhatsApp 链接」，点链接会自动打开 WA 并预填消息。

🏢 他们是什么样的公司

  • 公司类型: 发电机经销商、进口商、电力工程公司、EPC承包商
  • 主要城市: Lagos · Port Harcourt · Abuja · Kano · 各州中心城市
  • 公司规模: 年收入 ₦50M–₦5B (约 $30K–$3M USD)
  • 决策人: MD · 进口经理 · 采购负责人 · 业主
  • 采购频率: 每年 2-10 台，批量买家可达 20-50 台
  • 付款偏好: T/T (电汇) 或 LC (信用证)

😤 什么在驱动他们的需求

  • 利润被中国廉价品压缩 → 需要知名品牌保竞争力
  • 客户指定 Cummins/CAT → 但新机价格太高
  • EPC 项目时间紧 → 需要快速到货，不能等工厂
  • 地方经销商被 Lagos 中间商卡脖子 → 需要直接货源
  • 旧发电机队老化 → 需要合理成本刷新机队

✓ 高价值客户信号

  • ✓ 已在经营 Cummins 或 CAT 品牌
  • ✓ 有进口清关基础设施和经验
  • ✓ EPC 项目管线充足 或 计划扩充租赁机队
  • ✓ 主动抱怨现有供应商价格高或交期慢
  • ✓ 位于 Lagos、Port Harcourt 或主要商业中心

✗ 降低优先级的信号

  • ✗ 主要做维保服务，采购量极小
  • ✗ 体量过小，资金周转能力不足
  • ✗ 首次进口，无清关经验
  • ✗ 重心在消费级小型发电机 (<100kVA)
